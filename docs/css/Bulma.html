修饰符语法

何为修饰符语法，你可以理解为一个方法的接口，你可以在元素上加各种各样的接口

大多数 Bulma 元素都有可 选择 的样式，你只需要添加一个 修饰符 即可应用样式。他们都以 is- has- 开头.

颜色

  1. is-primary
  2. is-link
  3. is-info
  4. is-success
  5. is-warning
  6. is-danger
